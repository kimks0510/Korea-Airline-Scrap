# AGENTS.md

## 프로젝트: 대한항공 객실승무원 준비용 일일 뉴스 스크랩

## 목적
대한항공 객실승무원 지원·면접에 필요한 회사, 안전, 서비스, 노선, 통합 이슈를 매일 축적한다. 결과물은 Markdown 원본과 모바일 HTML 사이트로 보관한다.

## 매일 수집 범위
최근 24~48시간을 우선하며 각 항목은 최대 3건만 수집한다.

1. 대한항공·아시아나항공 통합, 조직, 경영
2. 항공 안전, 보안, 운항, 객실 안전
3. 기내 서비스, 고객 경험, 스카이패스
4. 노선, 항공기, 제휴·공항 서비스
5. 실적과 항공산업 환경
6. 채용, 교육, 조직문화, ESG

해당 뉴스가 없으면 `해당 없음`으로 쓰고 억지로 채우지 않는다. 대한항공 뉴스룸·공시·정부기관 등 1차 출처를 우선하며 모든 항목에 검증 가능한 링크를 붙인다. 기사 원문은 복사하지 않고 자기 언어로 요약한다. 추정이나 전망은 명확히 표시한다.

## 출력
매일 `output/YYYY-MM-DD-korean-air.md`에 저장한다. 오류는 `output/errors.log`에 추가한다. 숫자 범위에는 물결표 대신 `–`를 사용한다.

문서 순서는 다음과 같다.

1. 제목과 수집 기준 시각
2. 오늘의 3줄 총평
3. 누적 인사이트
   - 최근 7일: 반복되는 변화, 객실승무원 관점의 의미, 면접에 쓸 한 문장
   - 최근 30일: 구조적 흐름, 기회·위험, 다음 달 관찰 포인트
   - 기록이 7일/30일에 못 미치면 실제 축적 기간을 명시
4. 기사 스크랩
5. 오늘의 면접 연결 포인트
6. 초보자 용어 주석

각 기사는 아래 형식을 따른다.

```markdown
### [태그] 기사 제목
- 날짜: YYYY-MM-DD
- 핵심 내용: 사실과 숫자 중심 2–4문장
- 객실승무원 준비 관점: 안전·서비스·협업·회사 이해와 연결한 1–2문장
- 면접에서 이렇게 연결: 과장하지 않은 말하기 예시 1문장
- 출처: [매체명](URL)
```

## 초보자 설명 원칙
AOC, RTO, MRO, IRBS, ESG 같은 약어는 첫 등장 때 한국어 뜻과 실제 업무 의미를 괄호로 설명하고, 문서 아래 용어 주석에도 다시 정리한다. 회사 내부 용어나 항공업계 전문어를 설명 없이 사용하지 않는다.
