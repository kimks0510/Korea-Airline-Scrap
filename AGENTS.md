# AGENTS.md

## 프로젝트: 대한항공 객실승무원 준비용 일일 뉴스 스크랩

## 목적
대한항공 객실승무원 지원·면접에 필요한 회사, 안전, 서비스, 노선, 통합 이슈를 매일 축적한다. 결과물은 Markdown 원본과 모바일 HTML 사이트로 보관한다.

## 매일 수집 범위
최근 24~48시간을 우선하며 각 항목은 최대 3건만 수집한다.

1. 대한항공·아시아나항공 통합, 조직, 경영
2. 항공 안전, 보안, 운항, 객실 안전
3. 기내 서비스, 고객 경험, 스카이패스
4. 노선, 항공기, 제휴·공항 서비스
5. 실적과 항공산업 환경
6. 채용, 교육, 조직문화, ESG

해당 뉴스가 없으면 `해당 없음`으로 쓰고 억지로 채우지 않는다. 대한항공 뉴스룸·공시·정부기관 등 1차 출처를 우선하며 모든 항목에 검증 가능한 링크를 붙인다. 기사 원문은 복사하지 않고 자기 언어로 요약한다. 추정이나 전망은 명확히 표시한다.

## 출처 신뢰도와 면접 활용 기준
- **1순위 원문:** 대한항공·아시아나항공 뉴스룸과 IR·공시·채용, 국토교통부·항공철도사고조사위원회·공항공사·정부기관, ICAO·IATA·항공기 제작사 등 공식 자료. 안전·통합·노선·실적의 숫자와 일정은 우선 이 자료로 확정한다.
- **2순위 주요 보도:** Reuters, Bloomberg, Financial Times, Wall Street Journal, Nikkei Asia, 연합뉴스 및 주요 경제지의 직접 취재 기사. 공식 발표에 없는 업계 배경과 경쟁 구도를 보완한다.
- **3순위 전문자료:** 항공산업 조사기관·공항 통계·안전 전문기관의 방법론과 기준일이 공개된 자료.
- 면접에서 활용할 핵심 주장은 1순위 원문 하나 또는 독립적인 신뢰도 높은 출처 두 곳 이상으로 교차검증한다.
- 익명 관계자 보도는 `공식 확인 전`으로 표시한다. 블로그·카페·커뮤니티·출처 불명 SNS·홍보성 재가공 콘텐츠는 근거로 사용하지 않는다.
- 기사마다 `출처 등급: 1차 원문 | 주요 보도 | 전문자료`를 표시하고, `면접 활용 신뢰도: 높음 | 보통 | 확인 필요`를 함께 적는다.

## 출력
매일 `output/YYYY-MM-DD-korean-air.md`에 저장한다. 오류는 `output/errors.log`에 추가한다. 숫자 범위에는 물결표 대신 `–`를 사용한다.

문서 순서는 다음과 같다.

1. 제목과 수집 기준 시각
2. 오늘의 3줄 총평
3. 누적 인사이트
   - 최근 7일: 반복되는 변화, 객실승무원 관점의 의미, 면접에 쓸 한 문장
   - 최근 30일: 구조적 흐름, 기회·위험, 다음 달 관찰 포인트
   - 기록이 7일/30일에 못 미치면 실제 축적 기간을 명시
4. 기사 스크랩
5. 오늘의 면접 연결 포인트
6. 초보자 용어 주석

각 기사는 아래 형식을 따른다.

```markdown
### [태그] 기사 제목
- 날짜: YYYY-MM-DD
- 핵심 내용: 사실과 숫자 중심 2–4문장
- 객실승무원 준비 관점: 안전·서비스·협업·회사 이해와 연결한 1–2문장
- 면접에서 이렇게 연결: 과장하지 않은 말하기 예시 1문장
- 출처: [매체명](URL)
- 출처 등급: 1차 원문 | 주요 보도 | 전문자료
- 면접 활용 신뢰도: 높음 | 보통 | 확인 필요
```

## 초보자 설명 원칙
AOC, RTO, MRO, IRBS, ESG 같은 약어는 첫 등장 때 한국어 뜻과 실제 업무 의미를 괄호로 설명하고, 문서 아래 용어 주석에도 다시 정리한다. 회사 내부 용어나 항공업계 전문어를 설명 없이 사용하지 않는다.
